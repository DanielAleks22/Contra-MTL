API de gestion des contraventions et violations version 1.0

/contrevenants

/contrevenants get

get

Obtient la liste des contraventions entre deux dates.

get /contrevenants

Obtient la liste des contraventions entre deux dates.

  • Request
  • Response

Query Parameters

  • du: required(date)

    Date de début au format ISO 8601.

  • au: required(date)

    Date de fin au format ISO 8601.

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: Contravention

  • etablissement: required(string)
  • nombre_contraventions: required(integer)

Example:

[
  {
    "etablissement": "Café ",
    "nombre_contraventions": 5
  },
  {
    "etablissement": "Boulangerie délices",
    "nombre_contraventions": 2
  }
]

HTTP status code 500

Body

Media type: application/json

Type: object

Properties
  • error: required(string)

Example:

{
  "error": "An error occured while fetching the data"
}

/establishments/violations/xml get

get

get /establishments/violations/xml

Obtient la liste des établissements avec le nombre de contraventions connues en format XML, triée par ordre décroissant du nombre de contraventions.

Body

Type: any

/establishments/violations/csv

/establishments/violations/csv get

get

Obtient la liste des établissements avec le nombre de contraventions connues en format CSV, triée par ordre décroissant du nombre de contraventions. Le fichier CSV est encodé en UTF-8.

get /establishments/violations/csv

Obtient la liste des établissements avec le nombre de contraventions connues en format CSV, triée par ordre décroissant du nombre de contraventions. Le fichier CSV est encodé en UTF-8.

  • Response

HTTP status code 200

Body

Media type: text/csv

Type: any

Example:

"etablissement","nombre_contraventions"
"Restaurant bouffe","12"
"Café de la Gare","8"

HTTP status code 500

Body

Media type: text/plain

Type: any

Example:

An error occurred fetching the data
